Информационные ресурсы

Какие сервисы использовать для аналитики криптовалют? Где найти исторические данные, чтобы построить торгового робота? Где есть удобные инструменты, чтобы делать технический анализ? На данной странице собраны лучшие сервисы, которые могут быть полезны инвесторам, трейдерам, аналитикам и другим участникам рынка. Дано неполное описание, какие данные можно там найти - это может натолкнуть вас на какие-то идеи

--- Наши локальные ресурсы ---

 
Наше хранилище локального архива
 		  Хранилище материалов с курсами, сохранённых публичных материалов, выгрузка каналов Telegram, наши материалы и домашние работы, электронные книги и т.п.. Отрабатывает только для включённых в локальную информационную экосистему рабочих станций и обращения через браузер Konquerror
 
Оперативный раздел локального хранилища
 		  Материалы, с которыми идёт оперативная работа, ещё не перенесённые в неизменяемый архив. Отрабатывает только для включённых в локальную информационную экосистему рабочих станций и обращения через браузер Konquerror
 

--- Ресурсы для получения данных через API ---

 
https://www.cryptocompare.com
 		  очень много метаданных по всем криптовалютам. Есть бесплатное API c уникальными данными. Рейтинги кошельков, бирж, пластиковых карт, майнинг пулов. Бесплатные исторические данные через АПИ
 
https://www.coingecko.com
 		  Уникальность этого сервиса в том, что тут можно анализировать активность сообщества в соц. сетях и активность на Github. Все в удобной единой таблице с рейтингом. Аналитика активности по криптовалютным сообществам в Reddit: кол-во комментариев, среднее кол-во пользователей онлайн, среднее кол-во горячих постов за час, кол-во подписчиков в Twitter и Facebook. Данные по репозиториям на Github: коммиты, кол-во заданных вопросо, закрытых вопросов, кол-во форков, наблюдателей, активных участников, общее кол-во нововведений за 4 недели. Статистика по запросам в Bing
 

--- Ресурсы для проведения технического анализа ---

 
https://www.tradingview.com
 		  технический анализ в один клик. Есть все необходимые индикаторы для профессиональных трейдеров. Можно разбивать график на любые таймфреймы. Есть система уведомлений. Чат. Система шаринга идей с другими пользователям
 
https://www.coinglass.com/tv/Binance_BTCUSDT
 		  технический анализ в один клик. Там, где tradingView ограничивает количество индикаторов - этот ресурс не ограничивает, как и биржи и наш продукт
 
https://charts.cointrader.pro/charts.html?coin=BITCOIN%3AUSD
 		  Аналог TV от энтузиаста
 

--- Ресурсы с добротным теоретическим материалом ---

 
https://timetotrade.ru
 		  Ресурс валютного трэйдера Виталия. В т.ч. методички по SmartMoney
 
https://pro_trading.ru
 		  Некто. В т.ч. материалы по Эллиоту и Вайхофу
 

--- Ресурсы общеаналитические ---

 
https://coinmarketcap.com
 		  отличный сайт для оценки основной информации по рынку, где можно найти данные по всем блокчейн активам (839 в данный момент). Есть API. Интересные выборки: общая капитализация криптоэкономики, доля Биткоина по отношению к остальным, список новых коинов на биржах, рейтинги Biggest Gainers и Biggest Loosers. Исторические снепшоты основной таблицы
 
http://mapofcoins.com
 		  красивая инфографика по истории криптовалют и протоколов. Есть хорошая структурированная выжимка по каждой технологии. Помогает разобрать во всем зоопарке технологий и альткоинов
 
https://icostats.com
 		  ресурс позволяющий отслеживать итоги капитализации прошедших ICO в динамике
 
https://cryptowat.ch
 		  Этакий агрегатор всех бирж. Удобно смотреть real-time данные: цену, объемы, order book, список совершенных сделок. Есть АПИ. Coinigy.com — более крутой инструмент, но он полностью платный (хотя есть триал). Через АПИ есть доступ к историческим данным OHLC по разным биржам. Поддерживаются только некоторые валютные пары (самые популярные). Есть инструменты для графического анализа. В платной версии можно создать аккаунт и торговать на многих биржах из одного терминала
 
http://monetarism.ru
 		  Известный сайт графиков коинтеграции цен. Недавно добавили графики по Биткоину — тысячи их
 
https://tradeblock.com
 		  Вообще, позиционируется как инструмент для enterprise трейдеров. Уникален тем, что предоставляет много инструментов для анализа Bitcoin и Ethereum блокчейнов: графики транзакций, комиссий, аналитика блоков, мемпул — все в одном месте. Плюс удобный калькулятор для майнинга Bitcoin, который рассчитывает доходность на 5 лет вперед
 
https://coin.dance
 		  прекрасная разнообразная статистика и инфографика по Биткоину в разных разрезах. Политика в Биткоине: легальный статус в странах, внутрення политика Segwit vs BU. Объемы торгов по разным странам, в том числе на Localbitcoins. Google Trends запросы по разным странам. Аналитика аудитории Биткоина: по полу, возрасту, интересам. Статистика по майнинг пулам Исчерпывающая статистика по блокам и нодам. Можно поиграть в покер
 
https://blockchair.com
 		  мощный поиск по блокчейну Биткоина. Позволяет искать по всем данным в блоке, которые только возможны
